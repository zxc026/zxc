О проекте PC Craft

Курсовая работа — веб-приложение для сборки персонального компьютера с проверкой совместимости компонентов.

📌 Реализованный функционал

  • Выбор процессора, материнской платы, видеокарты, ОЗУ, накопителя, блока питания.
  • Автоматическая проверка совместимости: сокет CPU/MB, тип RAM, достаточность мощности БП.
  • Сохранение собранных конфигураций в базу данных (сессия пользователя).
  • Загрузка ранее сохранённых сборок.
  • Каталог всех комплектующих с детальными страницами.
  • Единый адаптивный дизайн на CSS.

🛠️ Технологии

  • Python 3, Django 4/5
  • SQLite (легко переключается на PostgreSQL)
  • HTML5, CSS3, Font Awesome
  • Git, GitHub
